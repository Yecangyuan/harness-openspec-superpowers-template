# Multi Review Adapter

Use `harness/skills/multi-review/SKILL.md` as the source of truth.

Before reviewing:

1. Read `HARNESS.md`.
2. Read `harness/skills/multi-review/SKILL.md`.
3. Load only the reviewer reference files selected by that workflow.
4. Return findings first, ordered by severity.
